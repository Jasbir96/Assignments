Which of the following scenarios is most suitable for using event capture in JavaScript? A) Handling click events on specific child elements within a parent element and executing different actions based on the clicked element. B) Handling input events on a form and validating user input before it reaches the form submit event. C) Adding event listeners to dynamically created elements that are appended to the DOM. D) Attaching event handlers to the window object to handle browser resize events. ## Answer: B
