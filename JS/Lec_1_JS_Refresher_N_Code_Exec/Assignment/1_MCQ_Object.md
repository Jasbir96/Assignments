## Description:

```js
let cap = {
    name: "Steve",
    age: 34,
    isAvenger: true,
}
for (let key in cap) {
    console.log(key, " ", cap[key]);
}
```

## Options: 
 a.) error

 b.)  name Steve
      age  34
      isAvenger true

  c.) name undefined
     age  undefined
     isAvenger undefined

 d.) No of the above
